autojump

### 安裝 {#installing}

```
shell> brew install autojump
```

`.zshrc`
```
[ -f /usr/local/etc/profile.d/autojump.sh ] && . /usr/local/etc/profile.d/autojump.sh
```

```
shell> autojump -s
shell> autojump --purge
shell> autojump -i [WEIGHT]
shell> autojump -d [WEIGHT]
```
