`qpdf - tools for transforming and inspecting PDF files`

### 安裝 {#installing}

```console
shell> brew install qpdf
shell> apt install qpdf
```

```console
shell> qpdf --show-encryption --password=password
shell> qpdf --decrypt --password=password
```
